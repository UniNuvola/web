## Setup

Install python dependencies with [Poetry](https://python-poetry.org/)

```bash
poetry install
```

Then create a `.env` file to store the Vault credentials

```
VAULT_CLIENT_ID=
VAULT_CLIENT_SECRET=
```

## Run

Activate the virtuan environment and run the web server with:

```bash
poetry shell
python main.py
```
